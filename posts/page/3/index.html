docker 快速扩容

docker 快速扩容 以celery worker为例。当启动docker-compose up后，可以看到有一个celery-worker容器在运行。 如果要临时增加，可以使用--scale参数配合up指令，接受多个容器传递： docker-compose up --scale web=2 celery-worker=3 经测试，多个worker都获得了发送过来的任务。

docker 清理
