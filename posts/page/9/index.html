pip 离线安装

pip 离线安装 打包 注意，要在同平台打包，否则有些包不能正确安装。 在已有的环境中，一般是一个虚拟环境： pip freeze > pip-requirements.txt pip download -d pip-packages -r pip-requirements.txt，将提取的包下载到pip-packages文件夹中 安装 将pip-requirements.txt和pip-packages文件夹，拷贝到目标环境的同目录下 pip install –no-index –find-links=pip-packages -r pip-requirements.txt 参考 断网环境下一键安装 python3 离线安装包及其依赖
